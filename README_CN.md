# 标题颜色选择器 :)

[中文说明](README_CN.md)

一个美观且直观的标题颜色选择器小组件，适用于 Trilium Notes。

> 💡 **小提示：如果你已经阅读了一段时间，不妨站起来伸展一下身体！**

# 📸 截图

简洁直观的界面

![整体界面](docs/main%20ui.webp)

实时预览 —— 颜色变化即时生效

![动态颜色变化](docs/dynamic%20color.gif)

点击笔记标题旁边的彩色圆点即可打开颜色选择器。你可以从预设颜色中选择，也可以使用完整颜色选择器。修改会实时应用到笔记标题和笔记树。

# 📥 安装方法

1. 从发布页面下载 zip 文件。
2. 在 Trilium 中右键点击笔记树，选择“导入（Import）”，取消勾选 `Safe Import`。
3. 重启 Trilium Notes，或使用 `Ctrl + R` 刷新界面。
4. 打开任意笔记，你会在笔记标题旁看到一个用于颜色选择的彩色圆点。
5. 尽情使用吧！

# 💡 配置说明

你可以轻松自定义该小组件：

* 打开导入的小组件中的 `config` 子笔记。
* 编辑 `presetColors` 数组，添加或修改你喜欢的颜色。
* 默认颜色是我最常使用的配色，你可以按个人喜好自由调整。

# 🤔 我为什么制作它？

我厌倦了每次都手动添加 `#color=orange` 标签。
我想要一种更快捷、更直观、更愉悦的方式给笔记添加颜色 —— 所以我制作了这个小组件。
