Click the color dot next to the note title to open the picker. Choose from your preset colors or use the full color picker. Changes apply in real-time to both the note title and the note tree.


# 📥 How to install

1. Download the zip file from [release page](https://github.com/Nriver/title-color-picker/releases).
2. Right click note tree in Trilium and click import, uncheck `Safe Import`.
3. Restart Trilium Notes or use `ctrl+r` to reload the interface.
4. Open any note, you will see a colored dot for color picker beside the note title.
5. Have fun.


# 💡 Configuration

You can easily customize the widget:

- Go to the `config` sub-note inside the imported widget.
- Edit the `presetColors` array to add or change your favorite colors.
- The defaults are the colors I use most often — feel free to personalize them!


# 🤔 Why I made this?

Tired of manually adding `#color=orange` labels every time.  
I wanted a fast, visual, and delightful way to color my notes — so I made this widget.
